01

Web Application Security and Vulnerability Remediation

OWASP Top 10 · SQL Injection · XSS · CSRF · RBAC

Web application security is one of the most in-demand skills in the cybersecurity job market and an area where I have direct, real-world experience. During my internship at Earth Viability Center, I identified and remediated live vulnerabilities including SQL injection, cross-site scripting (XSS), and cross-site request forgery (CSRF), all classified in the OWASP Top 10. I also designed and implemented a Role-Based Access Control (RBAC) system to reduce unauthorized privilege exposure across the application. After my internship, I extended this skill through a dedicated SQL Injection Testing Lab using DVWA and Mutillidae with SQLMap, which deepened my understanding of injection attacks from an offensive perspective.

Security Operations and Threat Detection

Splunk · Wireshark · Nmap · SIEM · Incident Response

Security operations is the day-to-day work of most cybersecurity teams, including monitoring systems, detecting anomalies, investigating alerts, and responding to incidents. I developed this skill through coursework and hands-on lab work using Splunk for SIEM log analysis and alerting, Wireshark for packet capture and network traffic analysis, and Nmap for host discovery and port scanning. Each tool covers a different phase of the security operations workflow: Splunk for detection, Wireshark for analysis, and Nmap for understanding network exposure from an attacker's perspective.
