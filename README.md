# Michael Walker — IDS 493 ePortfolio

Cybersecurity portfolio for IDS 493 at Old Dominion University.  
Built as a static HTML/CSS site hosted on GitHub Pages.

## 🚀 How to Deploy (GitHub Pages)

### Step 1 — Create a GitHub repo
1. Go to [github.com](https://github.com) and log in to your account (`mike0487`)
2. Click **"New"** (green button)
3. Name the repo: `portfolio` (or `ids493-portfolio`)
4. Set it to **Public**
5. Click **"Create repository"**

### Step 2 — Upload the files
1. In your new empty repo, click **"uploading an existing file"**
2. Drag and drop ALL files from this folder:
   - `index.html`
   - `skills.html`
   - `resume.html`
   - `essay.html`
   - `style.css`
3. Click **"Commit changes"**

### Step 3 — Enable GitHub Pages
1. Go to your repo's **Settings** tab
2. Click **"Pages"** in the left sidebar
3. Under **"Branch"**, select `main` and click **Save**
4. Wait 1-2 minutes, then your site will be live at:

```
https://mike0487.github.io/portfolio/
```

That's your URL to submit to Canvas.

## 📋 Things to Update Before Submitting

### Skills page (`skills.html`)
Each artifact card has a placeholder link (`href="#"`). Replace these with:
- **Artifact 1 (EVC Internship):** Upload your internship report as a PDF to the repo, then link it
- **Artifact 2 (SQL Injection Lab):** Already links to your GitHub — double check the URL
- **Artifact 3 (OWASP coursework):** Upload your course assignment as a PDF
- **AWS Cert:** Upload your cert PDF/image, link it
- **AWS IAM Lab:** Upload lab writeup
- **Cloud Architecture project:** Upload course project
- **Splunk / Wireshark / Nmap labs:** Upload lab reports

### Resume page (`resume.html`)
- Replace the "Download PDF Resume" link with a link to your actual LaTeX-generated PDF
- To add: upload your resume PDF to the repo (e.g., `Michael_Walker_Resume.pdf`)
- Then update the href in resume.html: `href="Michael_Walker_Resume.pdf"`

### How to upload files to your GitHub repo
1. Go to your repo on github.com
2. Click **"Add file" → "Upload files"**
3. Drag your PDFs/images in
4. Click **"Commit changes"**
